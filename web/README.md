## 安装

```sh
pnpm install
```

## 本地开发

```sh
pnpm dev
```

## 编译

```sh
pnpm build
```

## 本地预览

> 先执行编译再执行该命令

```sh
pnpm serve
```

## eslint检测

```sh
pnpm lint
```

## eslint修复

```sh
pnpm lint:fix
```

## prettier检测

```sh
pnpm format
```

## Git commit

```sh
pnpm cz
```
